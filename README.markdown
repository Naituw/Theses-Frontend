#Theses论文管理系统（前端, 毕业设计）
Theses论文管理系统的前端，基于Ember.js实现。
中山大学南方学院09级电软系毕业设计。

##Setup

1. 安装Nodejs
2. 使用r.js编译工具[进行编译](http://requirejs.org/docs/optimization.html#wholeproject)
3. 通过Nginx、Apache等HTTP服务器运行
